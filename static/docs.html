Chat

POST /chat
Headers:
  Authorization: Bearer <API_TOKEN>
  Content-Type: application/json
Body:
  {"message": "Hello Friday!"}

Try it in the browser chat.

Upload

POST /data/upload  (multipart/form-data)
Headers:
  Authorization: Bearer <API_TOKEN>
Parts:
  file   = (binary)
  notes  = optional text
